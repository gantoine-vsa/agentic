# AI SDK

A set of Agentic AI code examples using **Vercel AI SDK**

![image info](./images/orchestrator.png)

## Setup

```shell
pnpm i
```

### Maintenance

```shell
# keep deps up-to-date
pnpm up --latest
```

## Developer Workflow

### Lint

```shell
pnpm run lint
```

### Run

```shell
pnpm run enum
pnpm run array
pnpm run agent
pnpm run evaluator
```

### Build

```shell
pnpm run build
```
